# Detecção de Cheaters usando Aprendizado de Máquina
Este repositório contém o projeto final da matéria Inteligência Artificial.

O dataset é publico, e está disponível em: https://www.kaggle.com/datasets/emstatsl/csgo-cheating-dataset 

O projeto detalha a utilização de 3 algoritmos de machine learning(Random Forest, CNN e LSTM) para a detecção de cheaters(trapaceiros) no jogo CS:GO.
